# An E-Commerece Application (Offline) that have the following features:
- View as seller
  - Login
    - Add product
    - Remove Products
    - Display products
    - Back to MainMenu
  - Register
- View as customer
    - Buy Products
    - View Products 
    - Back to MainMenu
- Clear Data
    - Product Data
    - Seller Data 
- Exit

It Also have a simple JavaFX interface with all the alerts and exceptions
